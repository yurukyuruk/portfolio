Overview

The project involved designing a user interface for an event application called Eventful, which aimed to enhance user engagement by connecting individuals with shared interests. The app allows users to create, discover, and manage events based on personal preferences. It also provides users with tools to make reservations and efficiently manage their schedules. The target audience includes individuals looking for a streamlined experience in finding and attending events that cater to their personal interests.

Problem

Existing event management apps often suffer from poor navigation and a lack of user-centered features, leading to a frustrating user experience. Users found it difficult to find relevant events or manage them effectively due to complicated filtering systems and limited customization options. Additionally, key features such as event creation, profile management, and participation tracking were either missing or difficult to use. These issues diminished user engagement and discouraged people from using such platforms regularly.

Solution

To solve these issues, the Eventful app was designed with a focus on simplicity and usability. A clean, modern design with a strong emphasis on personalization allows users to easily navigate the app and find events that align with their interests. Key improvements included intuitive navigation, comprehensive filtering options, and detailed event management features. Usability testing played a critical role in identifying pain points, such as inconsistent button designs and the lack of certain features (e.g., calendar integration and event host information), leading to iterative improvements. The final product includes event creation tools, personalized recommendations, and the ability to manage event participation with ease. This approach ensured that the app not only addressed the user pain points but also delivered a user-friendly, engaging experience.

Research

After analyzing various event apps on the market, such as Meetup and Eventbrite, we assessed their strengths and weaknesses.

⟣ Our findings indicated that there is a demand for a platform that is easy to navigate, visually appealing, and equipped with comprehensive event management tools. A key pain point identified was the complicated navigation present in existing apps.

  • Site Map

    In the app, the site map is designed to ensure a smooth user experience, focusing on intuitive navigation and essential features. The app is structured around four key sections:

    1. Home:

      This is the main hub where users can search for events, filter results by category, and receive personalized, location-based recommendations.

    2. User Profile:

      Users can manage their personal information, such as updating their "About Me" section, selecting "Interests" to refine event suggestions, and managing participation in "My Groups."

    3. My Events:

      This section organizes the user's events into "Upcoming" and "Past Events," allowing them to easily track and revisit events they’re attending or have attended.

    4. Create New Event:

      Here, users can quickly create an event by filling in the details, such as name, description, date, time, and location.

    This site map emphasizes simplicity and user-centered design, ensuring that users can easily discover, participate in, and create events, making it an engaging platform for event management.

  • User Flow

    The user flow of my event app is designed to be intuitive, guiding users from discovering events to creating their own seamlessly. Starting from the Home Screen, users can engage in Event Discovery by either choosing an event from the main screen or navigating to the search function. They can Apply Filters to refine their search, view event details, and easily take action to Attend the Event.

    In the My Events section, users can view a list of Upcoming and Past Events, check details, and revisit those they have attended. The app also facilitates easy event creation, where users can fill out the necessary information, upload photos, and publish the event with just a few taps.

    This streamlined flow ensures users can effortlessly discover, attend, and create events, making event management an engaging experience.

Design

  • Style Guide

    Our design follows a clean, modern aesthetic, focusing on usability and visual appeal. The typography uses the Nunito font, creating distinct styles for headings and body text. The color palette, featuring vibrant orange and deep teal, was chosen to create a bold, energetic contrast that captures attention while maintaining a sense of balance and professionalism. Key UI elements include large, rounded primary buttons and minimalistic, monochrome icons for navigation. This cohesive design ensures clarity, easy navigation, and a consistent user experience.

    The first version of the design included all major features, such as event creation, profile management, and event discovery. Interactive elements were added to simulate real user interactions.
